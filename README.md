> **[N] 김소윤 Kim Soyoon**

>
> 안녕하세요, 긍정의 에너지로 세상을 바꾸고 싶은 가천 코코네스쿨 8기 김소윤입니다.
> 저의 취미는 맛있는 빵집 찾아다니기, 그림 그리기, 그리고 러닝입니다. 언뜻 공통점이 없어 보이지만, 저에게는 **'몰입의 즐거움'**이라는 하나의 맥락으로 이어집니다.
> 특히 빵을 너무나 사랑해서 시작한 하프 마라톤 완주 도전은, 제 스스로 목표를 설정하고 끝까지 밀어붙이는 **끈기**가 있다는 것을 증명하는 계기가 되었습니다.
> 이 전 과정을 유튜브 콘텐츠로 기록하며 **기록의 힘**도 몸소 배웠습니다.
> 저는 창업의 핵심이 **'문제를 깊게 파고들어 진짜 본질을 찾고, 이를 빠르게 검증하는 것'** 에 있다고 믿습니다.
> 단순히 상상에 머무는 아이디어가 아니라,'진짜 데이터'로 증명된 비즈니스 모델을 만들고자 합니다.
> 동시에, 문제 해결에 필요한 디자인과 콘텐츠를 직접 제작하며 상상을 현실로 빠르게 시각화하여 팀의 실행 속도를 높이는 데 기여하겠습니다.
> 긍정적이고 열정적인 성격으로 팀원을 연결하고, 같은 방향을 바라보는 팀원들과 함께 나아가고 싶습니다.
